### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/saggonzales/aloware.git 
   ```
2. Create database
3. Run composer update and migration
   ```sh
   composer update
   php artisan migrate
   ```
4. Install NPM packages
   ```sh
   npm install
   ```
5. Require vue-resource
   ```sh
   npm install --save vue-resource
   ```  
5. Require vue-resource
   ```sh
   php artisan key:generate
   ``` 

### Run the application
   ```sh
   php artisan serve
   ```
